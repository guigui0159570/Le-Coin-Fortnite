Descriptif

Fortnite est un jeu de tir et de stratégie, il a été crée le 21 Juillet 2018 par EPIC GAMES, Le premier mode qui a été crée est "Sauver le Monde". Puis arrive le célèbre mode "Battle Royale" puis plus tard le mode "Créatif"
