Descriptif

Fortnite Sauver le monde est le première mode étant sortir. une version PVE dans laquelle le joueur est seul ou en équipe combat des vagues de zombie pour protéger les lieux, des objets, des personnes...

Les différents lieux

Dans sauver le monde il y a plusieurs villes déblocable seulont l'avancement dans le jeux :

  1. Fontainebois

    2. Fontainebois est la première carte que les joueurs débloquent dans Sauver le Monde. Les missions qui s'y trouvent nécessitent une puissance allant de 1 à 19. Les joueurs doivent compléter toutes les quêtes principales proposées pour cette carte pour pouvoir débloquer la carte suivante Villeplanche. On y trouve plusieurs endroits différents : villes, zones urbaines, prairies et campagnes, à découvrir. Les joueurs doivent dès lors apprendre à construire des structures, placer des pièges contre les vagues de Carcasses. Il s'agit de la carte du début du jeu pour tous les nouveaux joueurs PvE, il y a donc plusieurs missions à compléter faisant office de tutoriel.

  2. Villeplanche

    3. Villeplanche est la deuxième carte que les joueurs peuvent débloquer dans Sauver le Monde après avoir réalisé toutes les quêtes principales se trouvant à Fontainebois. La puissance recommandée est 15-46. Débloquer Villeplanche débloque également le second Arbre de Talents et vous permet d'avoir la possibilité d'améliorer vos héros & schémas au au tier 2.

  3. Morne-la-Vallée

    4. Morne-la-Vallée est la troisième carte à débloquer dans Sauver le Monde . La puissance recommandée est 40-70. Actuellement, une première partie de quête a été ajouté pour cette carte. Depuis que Epic Games a donné la possibilité de réaliser les défenses de bouclier antitempête indépendamment des quêtes principales, il est possible de débloquer la quatrième carte Pic Hardis au bout de la 6ème défense de bouclier réussie. Morne-la-Vallée est particulier car c'est la première localisation à proposer un combat de boss: Le Roi de la Tempête.

  4. Pics Hardis

    5. Pics Hardis est la quatrième et dernière carte à débloquer dans Sauver le Monde. La puissance recommandée est 76-100. Il n'y a actuellement aucun scénario disponible pour cette carte qui est encore en dévelopement.
